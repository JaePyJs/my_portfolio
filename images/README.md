# Pixel Art Assets

This directory contains pixel art assets for the arcade portfolio.

## Required Assets

1. **pixel-character.png** - A pixel art character representing you
2. **about-bg.png** - Background for the About screen
3. **skills-bg.png** - Background for the Skills screen
4. **projects-bg.png** - Background for the Projects screen
5. **contact-bg.png** - Background for the Contact screen
6. **project1.png** - Pixel art representation of Discord Bot project
7. **project2.png** - Pixel art representation of Reddit Scraper project
8. **project3.png** - Pixel art representation of Portfolio project

## Creating Pixel Art

You can create pixel art using tools like:

1. [Piskel](https://www.piskelapp.com/) - Free online pixel art editor
2. [Aseprite](https://www.aseprite.org/) - Paid pixel art editor (highly recommended)
3. [Pixilart](https://www.pixilart.com/) - Free online pixel art editor

## Resources for Pixel Art

- [Lospec](https://lospec.com/) - Pixel art resources and palettes
- [OpenGameArt](https://opengameart.org/) - Free game assets including pixel art
- [itch.io](https://itch.io/game-assets/free/tag-pixel-art) - Free pixel art assets

## Recommended Dimensions

- Character: 32x32 or 64x64 pixels
- Background images: 320x240 pixels
- Project icons: 64x64 pixels

## Color Palette

Consider using a consistent color palette for all your pixel art assets. Some popular pixel art palettes include:

- [DawnBringer's 32 Color Palette](https://lospec.com/palette-list/dawnbringer-32)
- [PICO-8 Palette](https://lospec.com/palette-list/pico-8)
- [Endesga 32](https://lospec.com/palette-list/endesga-32)

## Tips for Pixel Art

1. Use a limited color palette
2. Focus on silhouettes and readability
3. Use dithering for gradients
4. Keep it simple - every pixel matters
5. Use reference images for inspiration
